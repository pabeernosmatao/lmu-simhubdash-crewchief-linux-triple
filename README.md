# LMU Linux Triple Monitor Guide (ES / EN)

Guide for setting up Le Mans Ultimate on Linux with SimHubDash and CrewChief, including a working triple monitor configuration (7680×1440, no Gamescope).

---

## 🌍 Languages

- 🇪🇸 Spanish → `Spanish/`
- 🇬🇧 English → `English/`

---

## 🚀 What this project includes

- Le Mans Ultimate (LMU) on Linux (CachyOS + KDE Plasma)
- SimHubDash telemetry, dashboards and overlays
- CrewChief integration
- Triple monitor setup (7680×1440 @ 144Hz)
- Wayland support
- No Gamescope required

---

## ⚠️ Notes

- Tested on CachyOS + KDE Plasma
- Designed for Wayland systems
- Focused on triple monitor racing setups
